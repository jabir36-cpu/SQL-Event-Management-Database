# SQL-Event-Management-Database
This is a mini event management system using MySQL and dataset are artificialy created for the purpose of practising. 

## Features
- Tables: Venues, Organizers, Events, Attendees
- Relationships with foreign keys
- Queries to list events, attendees, and revenue
- View and stored procedure examples

 ## Sample Queries
- List all events with venue and organizer
- Count attendees per event
- Calculate total revenue per event
- Get upcoming events
- Stored procedure to filter events by organizer
